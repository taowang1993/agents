# Exa Search — Pi Skill

A web search skill for Pi using the Exa API. Supports highlights, full text, LLM summaries, deep research, and structured outputs.

## Setup

Requires an Exa API key from https://dashboard.exa.ai/api-keys

1. Create an account at https://exa.ai
2. Get your API key from the dashboard
3. Add to your shell profile (`~/.zshrc`):
   ```bash
   export EXA_API_KEY="your-api-key-here"
   ```
4. Install dependencies (run once):
   ```bash
   cd {baseDir}
   npm install
   ```
